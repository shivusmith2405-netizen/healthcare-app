# AI Healthcare Web App

A full-stack healthcare web application built using Flask, MySQL, and JavaScript.

## Features

- Doctor search system
- Appointment booking
- Health dashboard with charts
- AI health assistant
- Health reels
- Bootstrap UI

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- Bootstrap

Backend:
- Python Flask
- REST APIs

Database:
- MySQL
